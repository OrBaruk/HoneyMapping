Initial prototype implementation of HoneyMapping
------------------

**Requirements:**
-	django
-	python
-	python-dev
-	geoip

**Usage:**
-	To run server:
	```
	python manage.py runserver 127.0.0.1:8000
	```

-	logs are located at:
	```
	{project_folder}/data/logs.txt
	```

-	Main page html at:
	```
	{project_folder}/main/templates/main/index.html
	```
